# My Profile

## このリポジトリについて

このリポジトリは、私自身のキャリアやスキル、制作物、実績、目標などを整理・蓄積するための、個人用ナレッジベースです。

これまでの経験を記録し、今後の成長とともに更新していくことを目的としています。

また、ChatGPT、Codex、ClaudeなどのAIアシスタントと情報を共有し、一貫したサポートを受けられる環境を作ることも目的としています。

## このリポジトリの使い方

このリポジトリは、履歴書や職務経歴書のような「完成品」を保存する場所ではありません。

自分自身の経験や学び、考え方を整理し、必要に応じて取り出せる「素材集」として活用します。

新しい経験や制作物、学びがあれば、その都度このリポジトリに追記し、継続的に育てていきます。
