# My Profile

## このリポジトリについて

このリポジトリは、私自身のキャリアやスキル、制作物、実績、目標などを整理・蓄積するための、個人用ナレッジベースです。

これまでの経験を記録し、今後の成長とともに更新していくことを目的としています。

また、ChatGPT、Codex、ClaudeなどのAIアシスタントと情報を共有し、一貫したサポートを受けられる環境を作ることも目的としています。

## このリポジトリで管理する内容

- [`00-project-rules.md`](00-project-rules.md) … プロジェクトの運用ルール
- [`10-profile.md`](10-profile.md) … 基本プロフィール
- [`15-environment.md`](15-environment.md) … 作業環境
- [`20-career.md`](20-career.md) … 職歴・経歴
- [`30-achievements.md`](30-achievements.md) … 実績・成果
- [`40-skills.md`](40-skills.md) … スキル一覧
- [`50-portfolio.md`](50-portfolio.md) … ポートフォリオ一覧
- [`60-goals.md`](60-goals.md) … 短期・中期・長期の目標
- [`70-preferences.md`](70-preferences.md) … 自分の考え方・AIへの希望
- [`80-interview.md`](80-interview.md) … 面接の記録・振り返り
- [`90-changelog.md`](90-changelog.md) … 更新履歴

## 情報の追加先

- 基本情報や自己紹介を更新する場合は、`10-profile.md` に追記する
- PC環境、使用ツール、操作上の前提条件を更新する場合は、`15-environment.md` に追記する
- 勤務先、担当業務、職務経験を追加する場合は、`20-career.md` に追記する
- 数値で示せる成果、受賞、業務改善の実績を追加する場合は、`30-achievements.md` に追記する
- 学んだ技術、使用経験、学習中のツールを追加する場合は、`40-skills.md` に追記する
- 制作したアプリケーションやツールを追加する場合は、`50-portfolio.md` に追記する
- 短期・中期・長期の目標を更新する場合は、`60-goals.md` に追記する
- 考え方、説明の好み、AIへの希望を更新する場合は、`70-preferences.md` に追記する
- 面接の質問、回答、振り返りを追加する場合は、`80-interview.md` に追記する
- リポジトリの運用ルールを変更する場合は、`00-project-rules.md` に追記する
- ファイルの追加や内容・構成の変更を行った場合は、`90-changelog.md` に記録する

複数のファイルに関係する情報は、それぞれの目的に合わせて必要なファイルを更新します。

## 機密情報の取り扱い

このリポジトリは公開されているため、次の情報は記録しません。

- パスワード、APIキー、アクセストークン、秘密鍵などの認証情報
- 住所、電話番号、メールアドレス、生年月日など、公開する必要のない個人情報
- 勤務先、取引先、顧客などに関する非公開情報や、守秘義務の対象となる情報
- 他者の氏名、連絡先、画像など、本人の許可を得ていない情報

個人や組織を特定できる情報を記録する必要がある場合は、公開して問題がないか確認し、必要に応じて匿名化・一般化します。

コミットまたはAIアシスタントと共有する前に、変更内容に機密情報が含まれていないか確認します。誤って認証情報を公開した場合は、ファイルから削除するだけでなく、該当する認証情報を直ちに無効化・再発行します。

## このリポジトリの使い方

このリポジトリは、履歴書や職務経歴書のような「完成品」を保存する場所ではありません。

自分自身の経験や学び、考え方を整理し、必要に応じて取り出せる「素材集」として活用します。

新しい経験や制作物、学びがあれば、その都度このリポジトリに追記し、継続的に育てていきます。
