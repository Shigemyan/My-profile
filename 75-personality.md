# Personality

## 診断結果

2026年7月14日に16Personalitiesで実施した診断では、**INFP-T（仲介者）** という結果だった。

| 特性 | 結果 |
| --- | --- |
| エネルギー | 94% 内向型 |
| 意識 | 96% 直感型 |
| 性質 | 72% 感情型 |
| 戦術 | 57% 探索型 |
| アイデンティティ | 54% 激動型 |

診断結果の確認ページ：

- [16Personalities 診断結果](https://www.16personalities.com/ja/%E7%B5%90%E6%9E%9C/infp-t/x/v84gskec0)

---

## 自己理解の参考

今回の結果では、内向型と直感型の傾向が特に強く出た。一人で考えを深めながら、新しい可能性や改善方法を考えることを好む傾向の参考になる。

感情型の傾向は、判断するときに人への影響や納得感を大切にしやすいことを示す。探索型は、新しい方法を試しながら柔軟に進める傾向の参考になる。

---

## 活用方針

この診断結果は自己理解のための参考情報として扱う。能力、適職、対人関係の相性などを決めつけるものではなく、実際の経験や行動とあわせて振り返る。

AIアシスタントには、次のような作業スタイルを前提として共有する。

- 一人で考えを整理する時間を大切にする
- 新しい技術や仕組みの可能性を考えることを好む
- 相手への影響や納得感を考慮して提案を行う
- 最初から完璧を求めるより、試しながら改善する
