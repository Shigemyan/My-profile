# Environment

このファイルでは、普段使用しているPC環境や周辺機器、ショートカットキー、AIへ説明する際の前提条件を管理します。

## キーボード

Windows配列のキーボードを使用しています。

Mac標準配列ではなく、Karabiner-Elementsでキー配列を変更しています。

### キー配列

| キー表示 | 実際の役割 |
|----------|------------|
| Ctrl | Command（⌘） |
| Alt | Option（⌥） |
| Win | Control |

## AIへの説明ルール

ショートカットキーを説明する場合は、Mac標準の「⌘」「⌥」ではなく、実際のキーボードに表示されているキー名を優先して説明します。

### 例

- Ctrl + C
- Ctrl + V
- Ctrl + S
- Ctrl + Tab
- Win + Tab

必要に応じて、Mac標準のキー名（⌘、⌥、Control）を補足として併記します。
